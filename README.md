# TIRASI-HP

生音 Acoustic Live vol.5 のチラシをベースにした、編集可能なイベントHP / A4チラシ作成アプリです。

## できること

- イベント情報をWebページとして表示
- HPトップページとしてイベント概要・スケジュール・Open Mic・アクセスを表示
- HPからA4チラシへ移動できるミニプレビューを表示
- ブラウザ上でチラシ内容を編集
- 複数イベントをlocalStorage内で管理
- 既存イベントの複製・新規作成・削除・切り替え
- 色テーマを春・夏・秋・夜から選択
- 編集内容をlocalStorageに保存
- 編集内容をJSONで保存 / 読み込み
- タイムスケジュールの追加・削除・上下移動
- A4プレビューを表示
- A4に収まりにくい可能性がある場合の注意表示
- ブラウザの印刷機能からPDF保存

## 画面

- `/`：イベントHPトップページ
- `/editor`：編集画面
- `/flyer`：A4チラシプレビュー

## 編集データの扱い

この初期版ではクラウド保存を使わず、編集内容はブラウザのlocalStorageに保存されます。

イベント内容を他の端末や次回イベントへ移したい場合は、編集画面の「JSON保存」でファイルを書き出し、「JSON読込」で読み込めます。

複数イベント管理もブラウザ内に保存されます。vol.5を複製してvol.6を作るような使い方ができます。

## 開発

```bash
npm install
npm run dev
npm run build
```

## 今回の方針

この初期版では、外部API・認証・DB・課金・AI接続は使いません。  
編集内容はユーザーのブラウザ内にだけ保存されます。

## Vercel

Vercelでは以下の設定で動きます。

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
